# Application IoT ESP32 - MQTT - Node-RED

> **Projet académique** - Module Technologies IoT  
> ENSA Khouribga | Janvier 2025  
> **Équipe de 4 personnes**

---

## Objectif

Développer un système IoT complet pour la surveillance environnementale (température, humidité, pression) avec :
- Acquisition de données via capteurs ESP32
- Transmission temps réel via protocole MQTT
- Visualisation sur dashboard Node-RED

---

## Technologies

- **Hardware** : ESP32 DevKit, capteurs DHT22, BMP180
- **Protocole** : MQTT 3.1.1
- **Broker** : Mosquitto
- **Plateforme** : Node-RED
- **Simulation** : Wokwi
- **Langage** : C/C++ (Arduino), JavaScript
